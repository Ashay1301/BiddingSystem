About

Welcome to our bidding platform, a comprehensive project developed by Group 11 of the Spring 2024 CS 5200 class at Northeastern University. Our team, comprised of Han Min, Muqing Fan, Yiqing Zhao, Varun Kabra, and Ashay Amish Panchal, is delighted to introduce you to this platform. Crafted with diligence, our website is built on the robust Django framework, backed by a MySQL database. We have also leveraged the capabilities of Google Cloud for deploying our database, ensuring reliability and scalability. We appreciate your interest in our project and extend our sincerest thanks for visiting our website.

Available Routes

  • / The home page. Contains news.
  • about/ Holds details about the application.
  • register/ Allows users to register an account
  • login/ Allows users to login to an account
  • logout/ Allows users to logout and clear their session
  • marketplace/ Listing of placements from various companies.
  • marketplace/slug/ Detailed view for a placement within the marketplace.
  • my-bids/ User scope bid summary for available placements.
  • dashboard/ Basic analytics and animated charts for the database.

Features and specs

  • Authentication system - users can login, register and logout, passwords are securely hashed with PBKDF2 method.
  • Certain routes are protected with a login required decorator, giving security to administrators/owners. Unregistered users are prompted to login on the home page.
  • Dashboard include dynamic and animated charts to draw insights from database.
  • Data is modelled in ORM and automatically generated by Django to RDBS of choice.
  • Data included in application is randomly generated using the faker library
  • App uses materialize css framework borrowing from Google's material design principles
  • App developed mainly in Django framework and Javascript.
